## What is this?

This project showcases a **frontend-only CRUD UI generator** built with **TypeScript and Next.js**.

The generator takes a simple entity schema and produces:
- TypeScript types
- Predictable mock data
- A List screen
- A Form screen

This repository is **not the generator itself**.  
It is a **public showcase** with examples of the generated output and an explanation of how the tool works.

---

## Why this exists

Building CRUD UIs is repetitive and time-consuming, especially for:
- internal business systems
- admin panels
- backoffice tools

This generator focuses on removing **boring frontend repetition**, without:
- backend generation
- databases
- authentication
- hidden abstractions
- AI guesses

> It removes repetitive UI work.  
> It does not replace thinking.
